# HR-Analytics-Dashboard
End-to-end HR Analytics Dashboard built in Power BI to analyze employee attrition, workforce demographics, compensation, performance, training, and recruitment insights.
## Project Overview
This Power BI dashboard provides a comprehensive analysis of workforce data, helping organizations monitor employee performance, attrition, compensation, training effectiveness, and recruitment trends.

## Key Features
- Employee Overview Analysis
- Attrition Analysis
- Compensation & Salary Analysis
- Performance & Training Analysis
- Recruitment & Diversity Analysis
- Interactive Filters and Drill-Down Reports

## Tools Used
- Power BI
- DAX
- Excel

## Business Insights
- Identified employee attrition patterns
- Analyzed department-wise workforce distribution
- Evaluated salary and compensation trends
- Measured employee performance and training effectiveness
- Monitored recruitment and diversity metrics
