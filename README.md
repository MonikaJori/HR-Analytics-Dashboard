# HR Analytics Dashboard

## Project Overview

Developed a comprehensive HR Analytics Dashboard using Power BI to transform raw employee data into actionable business insights. The dashboard enable HR teams and management to monitor workforce performance, analyze attrition patterns, evaluate compensation structures, track employee training and performance, and assess recruitment effectiveness through interactive visualizations and KPI-driven reporting.

## Key Highlights

- Analyzed 10,000+ employee records
- Built 6 interactive dashboard pages
- Created KPI cards for workforce monitoring
- Performed Attrition Analysis and Employee Segmentation
- Developed Compensation & Salary Analytics
- Implemented Performance and Training Analysis
- Designed Recruitment & Diversity Insights
- Applied Data Modeling and DAX Calculations
- Enabled dynamic filtering and drill-through analysis

## Tools & Technologies

- Power BI
- DAX
- Excel
- Data Modeling
- Business Intelligence

## Business Impact

The dashboard helps organizations identify employee retention challenges, monitor workforce trends, evaluate compensation fairness, and support strategic HR decision-making through data-driven insights.
